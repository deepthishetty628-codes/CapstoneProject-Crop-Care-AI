# CapstoneProject-Crop-Care-AI
🌿 AI-powered Plant Disease Detection System using PyTorch, CNN, Flask, and the PlantVillage dataset. Detects plant diseases from leaf images through a user-friendly web application with GPU-accelerated deep learning and real-time predictions.*

 #🌿 CropCare AI

An AI-powered Plant Disease Detection System that leverages Deep Learning and Computer Vision to identify plant diseases from leaf images. The application uses a Convolutional Neural Network (CNN) trained on the PlantVillage dataset and provides real-time disease predictions through an interactive web interface.

#🎯 Project Objective

The goal of this project is to develop a GPU-accelerated image classification system capable of automatically detecting plant diseases from leaf images. The system helps in early disease identification, enabling better crop management and supporting smart agriculture initiatives.

✨ Key Features
🌱 Plant disease detection from leaf images
🧠 Deep Learning-based CNN model
⚡ GPU-accelerated training
📷 Image upload and prediction
📊 Accuracy, Precision, Recall, and Loss evaluation
🌐 Flask-powered web application
🎨 Modern and responsive UI/UX
🚀 End-to-end AI deployment pipeline
